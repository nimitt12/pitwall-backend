# F1 Backend Server

A simple Express server to fetch F1 data.

## Setup

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## API Routes

- `GET /constructors`: Fetches the 2026 constructor standings from Ergast API.
